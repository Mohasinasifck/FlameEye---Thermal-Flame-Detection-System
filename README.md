# Fire and Smoke Detection using YOLOv8

This project implements a **real-time fire and smoke detection system** using a trained YOLOv8 model.  
The system captures frames from a webcam and performs inference to detect fire and smoke in real time.

## Features

- Real-time fire and smoke detection
- YOLOv8 custom trained model
- Live webcam inference
- FPS display for performance monitoring
- Simple and lightweight implementation

## Technologies Used

- Python
- YOLOv8 (Ultralytics)
- OpenCV
- PyTorch
